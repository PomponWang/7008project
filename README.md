# Dataset
[MovieLens 1M Dataset](https://grouplens.org/datasets/movielens/1m/) has been used for this project. MovieLens is a rating dataset from the MovieLens website. MovieLens data sets were collected by the GroupLens Research Project at the University of Minnesota.


This data set consists of:  

ratings.dat -- 1,000,209 anonymous ratings (1-5) of approximately 3,900 movies made by 6,040 MovieLens users. Users are numbered consecutively from 1. This is a table of "UserID :: MovieID :: Rating :: Timestamp".  

users.dat -- Infromation of users. This is a table of "UserID :: Gender :: Age :: Occupation :: Zip-code".  

movies.dat -- Information about the movies. This is a table of "MovieID :: Title :: Genres", where Genres are pipe-separated and are selected from the following 18 genres: action | adventure | animation | children's | comedy | crime | documentary | drama | fantasy | film-noir | horror | musical | mystery | romance | sci-fi | thriller | war | western.

Movies can be in several genres at once.
